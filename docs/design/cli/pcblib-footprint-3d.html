CLI command design

pcblib-footprint-3d

Generates standalone 3D preview artifacts for one footprint from an Altium PCB library. This command is currently deferred from migration because the old implementation is broken.

Usage

altium-cruncher pcblib-footprint-3d footprint.PcbLib FOOTPRINT_NAME -o output/preview

Arguments

Inputs identify a .PcbLib, footprint name, and output directory. The public command remains listed for migration tracking but should not be treated as release-ready until repaired.

Output

Target output is a 3D preview and supporting report. Release work should either fix and test this behavior or remove the command from the public manifest for the first release.

Tests

No release-grade workflow test is present yet. The migration plan explicitly marks this command as broken and not to migrate until repaired.
