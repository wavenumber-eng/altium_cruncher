CLI command design

version

Reports the installed package version using the date-version release policy.

Usage

altium-cruncher version
altium-cruncher --version

Arguments

The command takes no positional arguments. The top-level --version flag is equivalent.

Output

Writes altium-cruncher <version> to standard output and exits with status 0.

Tests

L0 and L99 verify that CLI output matches package metadata and the release date contract.
