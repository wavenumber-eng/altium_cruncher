CLI command design

easyeda-review

Builds fixture-wide review artifacts comparing EasyEDA schematic source data to generated Altium schematic output.

Usage

altium-cruncher easyeda-review C2040.json C21190.json -o output/review
altium-cruncher easyeda-review --fixture-dir tests/assets/easyeda/api_responses --pattern C*.json

Arguments

Inputs are saved EasyEDA JSON files or a fixture directory plus pattern. Pin policy flags match the schematic importer. The command requires the optional easyeda dependency extra.

Output

Outputs include per-case generated Altium libraries, EasyEDA source previews, review SVG, review HTML, and JSON reports under the selected output directory.

Tests

Optional EasyEDA tests validate multi-case schematic review generation and preview rendering with saved public JSON fixtures.
