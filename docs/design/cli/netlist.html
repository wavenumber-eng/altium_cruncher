CLI command design

netlist

Generates machine-consumable netlist JSON from Altium schematic or project inputs.

Usage

altium-cruncher netlist project.PrjPcb -o output/netlist
altium-cruncher netlist schematic.SchDoc

Arguments

file accepts .SchDoc or .PrjPcb. When omitted, the command attempts project auto-detection in the current directory. -o selects the output directory.

Output

The command writes schema-versioned JSON containing nets, components, pins, and connectivity. Future output naming should use the shared output path template resolver.

Tests

L3 verifies Hydroscope project netlist JSON generation and confirms a substantial net count. Additional contract tests should pin the JSON schema.
