easyeda-footprint-review

Builds fixture-wide review artifacts comparing EasyEDA footprint source data to generated Altium footprint output.

Usage

altium-cruncher easyeda-footprint-review C963370.json C266603.json -o output/review
altium-cruncher easyeda-footprint-review --fixture-dir tests/assets/easyeda/api_responses --pattern C*.json

Arguments

Inputs are saved EasyEDA JSON files or a fixture directory plus pattern. Policy flags match the footprint importer where needed. The command requires the optional easyeda dependency extra.

Output

Outputs include per-case Altium artifacts, EasyEDA source previews, review SVG, review HTML, and case reports under the selected output directory.

Tests

Optional EasyEDA tests validate multi-case footprint review generation from saved public JSON fixtures with altium-cruncher[easyeda] installed.
