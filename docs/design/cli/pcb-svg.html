CLI command design

pcb-svg

Renders PCB layer, board, outline, and assembly-view SVG artifacts from Altium PCB documents or projects.

Usage

altium-cruncher pcb-svg board.PcbDoc --export layers --layers top,bottom -o output/pcb-svg
altium-cruncher pcb-svg project.PrjPcb --config pcb-svg.json

Arguments

file accepts .PcbDoc or .PrjPcb. Config and CLI options control selected documents, view sets, export mode, layers, colors, drill rendering, metadata, clipping, scale, and output cleaning.

Output

The command writes SVG files for configured views. Planned work should formalize the pcb-svg.json contract and add stronger assembly-view HLR/geometer coverage.

Tests

L3 verifies Hydroscope PCB SVG generation from a copied PcbDoc and explicit config. Unit tests cover view selection and config behavior.
