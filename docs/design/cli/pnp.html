CLI command design

pnp

Exports pick-and-place placement data from Altium projects, currently as CSV or JSON.

Usage

altium-cruncher pnp project.PrjPcb --format json -o output/pnp
altium-cruncher pnp project.PrjPcb --format csv

Arguments

file is a .PrjPcb. --format selects CSV or JSON. Planned options include XLSX, JLC CPL, units, top/bottom grouping, designator ordering, variants, and shared BOM/PnP component filtering.

Output

The command writes placement data under the output directory. Planned output should share raw component data and alias handling with BOM and support config-driven naming.

Tests

L3 verifies Hydroscope JSON generation. Planned tests should use node_test_array for hierarchy, variant B4, sorting, and JLC CPL behavior.
