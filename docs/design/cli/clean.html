CLI command design

clean

Normalizes Altium schematic, schematic-library, and PCB-library style details with JSON-configured rules.

Usage

altium-cruncher clean input.SchDoc --config altium-clean.json -o output
altium-cruncher clean project.PrjPcb --backup
altium-cruncher clean footprint.PcbLib --config pcblib-clean.json

Arguments

file accepts project, schematic, schematic-library, or PCB-library files. --config selects rule JSON or creates a template when missing. --backup and --backup-path control source preservation. -o selects the output path or directory.

Output

The command writes normalized Altium files or config templates. Future work should add a formal config contract and a preview workflow for reviewing rule effects before mutation.

Tests

L3 verifies template creation without mutating committed Hydroscope assets. Existing unit tests cover schematic and PCB-library clean helpers; release signoff must add richer config-contract tests.
