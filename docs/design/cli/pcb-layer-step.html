pcb-layer-step

Generates a colored STEP model for one selected PCB layer using Altium PCB geometry and geometer planar extrusion.

Usage

altium-cruncher pcb-layer-step board.PcbDoc --layer bottom --copper-color #3D85C6 --outline-color #CCCCCC -o output/step
altium-cruncher pcb-layer-step project.PrjPcb --doc board.PcbDoc --config pcb-layer-step.json

Arguments

file accepts .PcbDoc or .PrjPcb. Options select layer, project context, board document, colors, extrusion thickness, outline behavior, hole clipping, polygon inclusion, and fusion.

Output

The command writes a STEP file plus JSON manifest. The manifest records layer, color options, geometry counts, backend, source input, and generated filenames.

Tests

L3 uses a minimized cricket-node PcbDoc to verify bottom-layer command behavior and color propagation. Unit tests cover JSON/JSONC config loading, layer selectors, arc sampling, and geometer request shape.
